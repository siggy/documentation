reports

Report problematic users to your moderators.

Responses are not currently documented.

post
File a report

https://mastodon.example/api/v1/reports

Returns: Report
OAuth: User token + write:reports
Version history:
1.1 - added
2.3.0 - add forward parameter

Request

Headers
Authorization
required
string
Bearer <user token>
Form Data Parameters
account_id
required
string
ID of the account to report
status_ids
optional
array
Array of Statuses to attach to the report, for context
comment
optional
string
Reason for the report (default max 1000 characters)
forward
optional
boolean
If the account is remote, should the report be forwarded to the remote admin?

Response

200: Success
